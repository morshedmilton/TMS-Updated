# Tournament Management System (TMS-Updated)

A comprehensive web application for managing gaming tournaments efficiently and effectively.

## Overview

The Tournament Management System enables users to create and manage gaming tournaments with automated scheduling, team registration, real-time match results, and tournament rankings.

## Features

- **User Authentication**: Secure registration and login with role-based access (Admin, Player, Team Captain)
- **Tournament Management**: Create, browse, and manage tournaments with various game types
- **Team Registration**: Players can form teams and register for competitive tournaments
- **Automated Scheduling**: Generate match schedules using round-robin algorithm
- **Result Management**: Real-time match result entry and tournament standings calculation
- **Game Catalog**: Admin interface for managing available games
- **User Management**: Account activation/deactivation and user administration

## Technology Stack

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: C# / ASP.NET (or your actual tech)
- **Database**: SQL Server
- **Version Control**: Git & GitHub

## User Roles

### Player
- Browse available tournaments
- View match schedules and results
- Manage personal profile

### Team Captain
- All Player functionalities
- Create and manage teams
- Register teams for tournaments
- Manage team roster

### Admin
- Create and manage tournaments
- Generate match schedules
- Enter match results
- Manage users and teams
